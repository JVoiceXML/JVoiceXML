Eclipse Plugin

There is an eclipse plugin available that allows to start and stop the JVoiceXML voice browser from eclipse using the server view.

The plugin can be installed manually by downloading the plugin and copying it to the plugin directory of your eclipse installation or by using the update mechanism of eclipse.

The URL of the update site is http://jvoicexml.sourceforge.net/eclipseupdates
