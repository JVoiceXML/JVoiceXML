Demos

JVoiceXML is shipped with some simple demo programs that should ease the way to get started.

Preconditions

  • Java 8 in the classpath

Starting the demos

The demos may require enabling of different implementation platform. Ensure that they are enabled before starting JVoiceXML. The following table lists the required implementation platforms per demo.

Demo Required implementation platforms
org.jvoicexml.demo.gotodemo jsapi20
org.jvoicexml.demo.helloworlddemo jsapi20
org.jvoicexml.demo.helloworldservletdemo jsapi20
org.jvoicexml.demo.inputdemo jsapi20
org.jvoicexml.demo.jtapidemo jsapi20 + jtapi callmanager
org.jvoicexml.demo.luis text + luis grammar
org.jvoicexml.demo.mixedinitiativedemo jsapi20
org.jvoicexml.demo.mmi.simpledemo jsapi20 + mmi callmanager
org.jvoicexml.demo.objecttagdemo jsapi20
org.jvoicexml.demo.recorddemo jsapi20
org.jvoicexml.demo.scriptdemo jsapi20
org.jvoicexml.demo.simpleinputdemo jsapi20
org.jvoicexml.demo.talkingheaddemo talking head
org.jvoicexml.demo.textdemo text
org.jvoicexml.demo.voicexmlcreationdemo none
org.jvoicexml.voicexmluni.demo text

On a command prompt or shell CD to the demo directory and run
gradlew -p <demo project directory> run
