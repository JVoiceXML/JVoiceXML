JVoiceXML Tag Support

JVoiceXML is in active development. Unfortunately not all tags are supported completly, but we are working on it. The list below gives you an impression about the current support. The good news: nearly all tags are supported initially. This enables you to develop VoiceXML simple application and they should behave "as expected". Note, that there are also some things going on behind simply reading the tags and passing them to some processing instance. Please, drop us a note if you have any problems using the voice browser or you discover that this description is wrong.
