Library dependencies

JVoiceXML libraries

JVoiceXML comprises the following jar files:

  • jvxml.jar
    This is the main jar, containing the VoiceXML interpreter
  • jvxml-xml.jar
    This jar contains all the XML related stuff
  • org.jvoicexml.implementation.jsapi10.jar
    This jar contains a basic implementation platform based on the JSAPI 1.0.
  • org.jvoicexml.processor.srgs.jar
    This jar contains an SRGS grammar processor.
  • org.jvoicexml.jndi.jar
    This jar contains the JNDI support for JVoiceXML which is needed for remote access.
  • org.jvoicexml.implementation.jsapi10-impl.jar
    This jar contains some demo implementations for the JSAPI 1.0 using some open source libraries. Currently we support FreeTTS as the TTS engine and sphinx 4 as the voice recognizer.
  • jvxml-jsapi2.0.jar
    This jar contains an unfinished implementation platform based on the JSAPI 2.0.
  • jvxml-jsapi2.0-impl.jar
    This jar contains some unfinished demo implementations for the JSAPI 2.0 using some open source libraries. Currently we support FreeTTS as the TTS engine and sphinx 4 as the voice recognizer.
  • jvxml-text.jar
    This jar contains a basic implementation platform for text based interaction.
  • jvxml-jtapi.jar
    This jar contains an unfinished implementation for jtapi 1.3.1 support
  • org.jvoicexml.jndi.jar
    This jar contains support remote access to the JVoiceXML server
  • jvmxl-client.jar
    jar containing the interfaces to create client applications.
  • jvmxl-client-text.jar
    jar containing the classes to access the text based implementation platform

Library dependencies

All libraries can be found in the $JVOICEXML_HOME/lib folder. You will need:

To create VoiceXML documents

  • jvxml-xml.jar

To run the interpreter

  • jvxml.jar
  • jvxml-xml.jar
  • org.jvoicexml.processor.srgs.jar
  • jvxml-client.jar
  • org.jvoicexml.jndi.jar (optional)
  • log4j 1.2.16 or above (http://logging.apache.org/log4j/)
  • commons libraries (http://jakarta.apache.org/commons/)
    • logging 1.1.1
    • pool 1.5.3
    • http client 4.0.1
    • codec 1.4
  • rhino 1.7R2 (http://www.mozilla.org/rhino/)
  • spring framework 2.5.6 (http://www.springframework.org/)
  • jchardet 1.1 (http://jchardet.sourceforge.net/)

Interpreter with JSAPI 1.0 support

  • org.jvoicexml.implementation.jsapi10.jar
  • org.jvoicexml.implementation.jsapi10-impl.jar
  • JSAPI 1.0 (http://java.sun.com/products/java-media/speech/)
  • FreeTTS 1.2.2 (SVN snapshot) (http://freetts.sourceforge.rtp.orgnet/) (This includes the JSAPI)
  • sphinx 4 (SVN snapshot) (http://cmusphinx.sourceforge.net/)

Interpreter with JSAPI 2.0 support (unfinished)

  • JSAPI 2.0 (must be downloaded from http://www.conversations.com/)
  • jvxml-jsapi2.0.jar
  • jvxml-jsapi2.0-impl.jar
  • FreeTTS 1.2.2 (SVN snapshot) (http://freetts.sourceforge.net/)
  • sphinx 4 (SVN snapshot)(http://cmusphinx.sourceforge.net/)
  • jlibrtp.jar (http://jlibrtp.org/)

Interpreter with Mary support

  • org.jvoicexml.implementation.mary.jar
  • maryclient.jar (http://mary.dfki.de)

Interpreter with MRCPv2 support

  • jvxml-mrcpv2.jar
  • cairo-client.jar
  • cairo-sip-0.2.jar
  • jain-sip-ri-1.2.jar
  • jain-sip-api-1.2.jar
  • nist-sdp-1.0.jar
  • concurrent-1.3.4.jar
  • mrcp4j-0.2.jar
  • zanzibar.jar
  • commons-lang-2.3.jar
  • MRCPv2 server
  • SIP phone

Interpreter with text based interface

  • jvxml-text.jar

To run a client

  • jvxml-client.jar
  • Rhino Scripting library
  • jvxm-xml.jar (optional if you want to author VoiceXML documents
  • jvxml-client-text.jar (optional, to access the text based implementation platform)

You can also download the third party libraries from their web sites.
